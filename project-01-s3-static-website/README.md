# Project 1: S3 Static Website Hosting

## Overview

This project demonstrates how to host a static website using Amazon S3.

## Objectives

- Create an S3 bucket
- Configure static website hosting
- Upload website files
- Configure bucket permissions
- Verify website accessibility

## AWS Services Used

- Amazon S3

## Skills Demonstrated

- Cloud Storage
- Static Website Hosting
- AWS Console Navigation
- Access Control
- Troubleshooting

## Steps Performed

1. Created an S3 bucket
2. Enabled static website hosting
3. Uploaded website files
4. Configured bucket permissions
5. Tested website functionality

## Outcome

Successfully deployed and accessed a static website hosted on Amazon S3.

## Lessons Learned

- How S3 can host static websites
- The importance of bucket permissions
- Basic AWS troubleshooting techniques
