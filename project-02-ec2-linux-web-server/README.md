# Project 2: EC2 Linux Web Server

## Overview

This project demonstrates how to launch an Amazon EC2 instance, connect through SSH, and deploy a web server using Nginx.

## Objectives

- Launch an EC2 instance
- Configure security groups
- Connect using SSH
- Install Nginx
- Verify web server accessibility

## AWS Services Used

- Amazon EC2
- Security Groups

## Linux Commands Used

- ssh
- sudo
- systemctl
- yum
- whoami
- df -h

## Skills Demonstrated

- Linux Administration
- Cloud Infrastructure
- Network Security
- Server Deployment
- Troubleshooting

## Steps Performed

1. Launched an Amazon Linux EC2 instance
2. Configured inbound security group rules
3. Connected via SSH
4. Installed Nginx
5. Started and enabled the Nginx service
6. Verified web server functionality through a browser

## Outcome

Successfully deployed a Linux web server on AWS and confirmed public access.

## Lessons Learned

- How EC2 instances operate
- Importance of security groups
- Basic Linux server management
- Troubleshooting connectivity issues
